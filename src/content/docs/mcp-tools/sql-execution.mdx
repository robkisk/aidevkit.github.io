## execute_sql

**Description:** Execute a SQL query on a Databricks SQL Warehouse.

**Parameters:**

| Parameter | Type | Required | Description |
|---|---|---|---|
| `sql_query` | `str` | Yes | SQL query to execute |
| `warehouse_id` | `str` | No | Optional warehouse ID. If not provided, auto-selects one. |
| `catalog` | `str` | No | Optional catalog context for unqualified table names. |
| `schema` | `str` | No | Optional schema context for unqualified table names. |
| `timeout` | `int` | No | Timeout in seconds (default: 180) |
| `query_tags` | `str` | No | Optional query tags for cost attribution (e.g., "team:eng,cost_center:701"). Appears in system.query.history and Query History UI. |
| `output_format` | `str` | No | Result format — "markdown" (default) or "json". Markdown tables are ~50% smaller than JSON because column names appear only once in the header instead of on every row. Use "json" when you need machine-parseable output. |

## execute_sql_multi

**Description:** Execute multiple SQL statements with dependency-aware parallelism.

**Parameters:**

| Parameter | Type | Required | Description |
|---|---|---|---|
| `sql_content` | `str` | Yes | SQL content with multiple statements separated by ; |
| `warehouse_id` | `str` | No | Optional warehouse ID. If not provided, auto-selects one. |
| `catalog` | `str` | No | Optional catalog context for unqualified table names. |
| `schema` | `str` | No | Optional schema context for unqualified table names. |
| `timeout` | `int` | No | Timeout per query in seconds (default: 180) |
| `max_workers` | `int` | No | Maximum parallel queries per group (default: 4) |
| `query_tags` | `str` | No | Optional query tags for cost attribution (e.g., "team:eng,cost_center:701"). |
| `output_format` | `str` | No | Result format — "markdown" (default) or "json". Markdown tables are ~50% smaller than JSON because column names appear only once in the header instead of on every row. |

## get_table_details

**Description:** Get table schema and statistics for one or more tables.

**Parameters:**

| Parameter | Type | Required | Description |
|---|---|---|---|
| `catalog` | `str` | Yes | Unity Catalog name |
| `schema` | `str` | Yes | Schema name |
| `table_names` | `List[str]` | No | List of table names or GLOB patterns (e.g., ["bronze_*", "silver_orders"]). If None, returns all tables in the schema. |
| `table_stat_level` | `str` | No | Level of statistics to collect: - "NONE": Schema only, no statistics - "SIMPLE": Row count and basic info (default) - "DETAILED": Column-level statistics including histograms |
| `warehouse_id` | `str` | No | Optional warehouse ID. If not provided, auto-selects one. |

## get_volume_folder_details

**Description:** Get schema and statistics for data files in a Databricks Volume folder.

**Parameters:**

| Parameter | Type | Required | Description |
|---|---|---|---|
| `volume_path` | `str` | Yes | Path to the volume folder. Can be: - "catalog/schema/volume/path" (e.g., "ai_dev_kit/demo/raw_data/customers") - "/Volumes/catalog/schema/volume/path" |
| `format` | `str` | No | Data format - "parquet", "csv", "json", "delta", or "file" (just list files). |
| `table_stat_level` | `str` | No | Level of statistics - "NONE", "SIMPLE" (default), or "DETAILED". |
| `warehouse_id` | `str` | No | Optional warehouse ID. If not provided, auto-selects one. |
