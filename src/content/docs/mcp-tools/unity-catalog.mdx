Browse and manage Unity Catalog objects, permissions, volumes, and governance policies.

### Example prompts

> "List all tables in my_catalog.my_schema"

> "Grant SELECT on this table to the data-analysts group"

> "Upload this CSV to /Volumes/catalog/schema/landing/data.csv"

---

## create_volume_directory

**Description:** Create a directory in a Unity Catalog volume.

**Parameters:**

| Parameter | Type | Required | Description |
|---|---|---|---|
| `volume_path` | `str` | Yes | Path for new directory (e.g., "/Volumes/catalog/schema/volume/new_folder") |

## delete_from_volume

**Description:** Delete a file or directory from a Unity Catalog volume.

**Parameters:**

| Parameter | Type | Required | Description |
|---|---|---|---|
| `volume_path` | `str` | Yes | Path to file or directory (e.g., "/Volumes/catalog/schema/volume/folder") |
| `recursive` | `bool` | No | Delete directory contents (required for non-empty dirs) |
| `max_workers` | `int` | No | Parallel delete threads (default: 4) |

## download_from_volume

**Description:** Download a file from a Unity Catalog volume to local path.

**Parameters:**

| Parameter | Type | Required | Description |
|---|---|---|---|
| `volume_path` | `str` | Yes | Path in volume (e.g., "/Volumes/catalog/schema/volume/data.csv") |
| `local_path` | `str` | Yes | Target local file path |
| `overwrite` | `bool` | No | Whether to overwrite existing local file (default: True) |

## get_volume_file_info

## list_volume_files

**Parameters:**

## manage_metric_views

**Description:** Manage Unity Catalog metric views: create, alter, describe, query, drop, and grant.

**Parameters:**

| Parameter | Type | Required | Description |
|---|---|---|---|
| `action` | `str` | Yes | "create", "alter", "describe", "query", "drop", or "grant" |
| `full_name` | `str` | Yes | Three-level name (catalog.schema.metric_view_name) |
| `source` | `str` | No | Source table/view (for create/alter, e.g., "catalog.schema.orders") |
| `dimensions` | `List[Dict[str, str]]` | No | List of dimension dicts for create/alter. Each has: - name: Display name (e.g., "Order Month") - expr: SQL expression (e.g., "DATE_TRUNC('MONTH', order_date)") - comment: (optional) Description |
| `measures` | `List[Dict[str, str]]` | No | List of measure dicts for create/alter. Each has: - name: Display name (e.g., "Total Revenue") - expr: Aggregate expression (e.g., "SUM(total_price)") - comment: (optional) Description |
| `version` | `str` | No | YAML spec version (default: "1.1" for DBR 17.2+) |
| `comment` | `str` | No | Description of the metric view (for create/alter) |
| `filter_expr` | `str` | No | SQL boolean filter applied to all queries (for create/alter) |
| `joins` | `List[Dict[str, Any]]` | No | Star/snowflake schema joins (for create/alter). Each dict: name, source, on (or using), joins (nested for snowflake) |
| `materialization` | `Dict[str, Any]` | No | Materialization config (experimental, for create/alter). |
| `or_replace` | `bool` | No | If True, uses CREATE OR REPLACE (for create, default: False) |
| `query_measures` | `List[str]` | No | Measure names to query (for query action) |
| `query_dimensions` | `List[str]` | No | Dimension names to group by (for query action) |
| `where` | `str` | No | WHERE clause filter (for query action) |
| `order_by` | `str` | No | ORDER BY clause, use "ALL" for ORDER BY ALL (for query action) |
| `limit` | `int` | No | Row limit (for query action) |
| `principal` | `str` | No | User/group to grant to (for grant action) |
| `privileges` | `List[str]` | No | Privileges to grant, default ["SELECT"] (for grant action) |
| `warehouse_id` | `str` | No | SQL warehouse ID (auto-selected if not provided) |

## manage_uc_grants

|---|---|---|---|

## manage_uc_objects

**Description:** Manage Unity Catalog namespace objects: catalogs, schemas, volumes, functions.

**Parameters:**

| Parameter | Type | Required | Description |
|---|---|---|---|
| `object_type` | `str` | Yes | "catalog", "schema", "volume", or "function" |
| `action` | `str` | Yes | "create", "get", "list", "update", or "delete" |
| `name` | `str` | No | Object name (for create) |
| `full_name` | `str` | No | Full qualified name (for get/update/delete). |
| `catalog_name` | `str` | No | Parent catalog (for list schemas/volumes/functions, or create schema) |
| `schema_name` | `str` | No | Parent schema (for list volumes/functions, or create volume) |
| `comment` | `str` | No | Description (for create/update) |
| `owner` | `str` | No | Owner (for create/update) |
| `storage_root` | `str` | No | Managed storage location (for catalog/schema create) |
| `volume_type` | `str` | No | "MANAGED" or "EXTERNAL" (for volume create, default: MANAGED) |
| `storage_location` | `str` | No | Cloud storage URL (for external volumes) |
| `new_name` | `str` | No | New name (for update/rename) |
| `properties` | `Dict[str, str]` | No | Key-value properties (for catalog create) |
| `isolation_mode` | `str` | No | "OPEN" or "ISOLATED" (for catalog update) |
| `force` | `bool` | No | Force deletion (default: False) |
