## get_serving_endpoint_status

**Description:** Retrieves the detailed status and configuration of a specific serving endpoint including its served models, traffic routing, compute configuration, and readiness state.

**Parameters:**

| Parameter | Type | Required | Description |
|---|---|---|---|
| name | string | Yes | Endpoint name to look up |

**Example prompt:**
```
Check the status and traffic config of the customer-support-agent endpoint
```

**Related skills:** [Model Serving](/aidevkit.github.io/skills/ai-ml/model-serving/)

## query_serving_endpoint

**Description:** Sends an inference request to a model serving endpoint. Supports chat completions, embeddings, and custom model input formats depending on the endpoint type.

**Parameters:**

| Parameter | Type | Required | Description |
|---|---|---|---|
| name | string | Yes | Endpoint name to query |
| messages | array | No | Chat messages for chat completion endpoints |
| input | object | No | Input payload for custom model endpoints |
| max_tokens | number | No | Maximum tokens in the response |
| temperature | number | No | Sampling temperature (0.0 to 2.0) |

**Example prompt:**
```
Send a test query to the customer-support-agent endpoint:
"How do I reset my password?"
```

**Related skills:** [Model Serving](/aidevkit.github.io/skills/ai-ml/model-serving/), [GenAI Agents](/aidevkit.github.io/skills/ai-ml/model-serving/genai-agents/)
