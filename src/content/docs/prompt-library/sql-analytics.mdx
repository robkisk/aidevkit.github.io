### Pipe Syntax

```
Rewrite this SQL query using Databricks pipe syntax (|>):
SELECT department, AVG(salary) as avg_salary
FROM main.hr.employees
WHERE status = 'active'
GROUP BY department
HAVING AVG(salary) > 100000
ORDER BY avg_salary DESC
```
