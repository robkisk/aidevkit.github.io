## Creating a Knowledge Assistant

```python
manage_ka(
    action="create_or_update",
    name="Company Policy Assistant",
    volume_path="/Volumes/catalog/schema/volume/policies",
    description="Answers questions about company policies and procedures",
    instructions="Answer concisely, cite the source document, and say 'I don't know' if unsure"
)
```

### Prerequisites

- Documents (PDF, text, or other files) stored in a Unity Catalog Volume
- Generate synthetic documents with the `databricks-unstructured-pdf-generation` skill if needed

### Auto-Examples

When `add_examples_from_volume=true` (default), examples are automatically extracted from JSON files in the volume and added once the endpoint is ONLINE.

### Operations

| Action | Purpose |
|--------|---------|
| `create_or_update` | Create new or update existing KA |
| `get` | Get KA details by tile_id |
| `find_by_name` | Look up KA by name |
| `delete` | Delete a KA |
