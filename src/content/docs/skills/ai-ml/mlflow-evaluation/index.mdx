## Overview

The MLflow Evaluation skill teaches Claude Code how to evaluate GenAI agents using MLflow 3's evaluation framework. It covers building evaluation datasets, creating custom scorers, running evaluations, analyzing traces for debugging, aligning LLM judges with domain expert feedback via MemAlign, and running automated prompt optimization with GEPA. Use it when measuring agent quality, comparing versions, or setting up production monitoring.

## Capabilities

- Run evaluations with `mlflow.genai.evaluate()` using built-in and custom scorers
- Build evaluation datasets from traces, manual examples, or Unity Catalog tables
- Create custom `@scorer` functions for project-specific metrics
- Use built-in scorers: Guidelines, Correctness, Safety, RetrievalGroundedness
- Analyze production traces for latency profiling, token usage, and error patterns
- Optimize retrieval context for cost and relevance
- Set up Unity Catalog trace ingestion and production monitoring
- Align LLM judges to match domain expert preferences with MemAlign
- Run automated prompt improvement with `optimize_prompts()` and GEPA

## Key Concepts

### Critical API Facts

- Use `mlflow.genai.evaluate()` (NOT `mlflow.evaluate()`)
- Data format: `{"inputs": {"query": "..."}}` (nested structure required)
- `predict_fn` receives `**unpacked kwargs` (not a dict)
- MemAlign is scorer-agnostic (works with float, bool, or categorical feedback)
- `optimize_prompts` requires MLflow >= 3.5.0

### Evaluation Workflows

| Goal | Workflow |
|------|---------|
| First-time setup | Build dataset, choose scorers, run evaluation |
| Production traces to dataset | Search traces, tag interesting ones, build dataset |
| Performance optimization | Profile latency, analyze tokens, apply optimizations |
| Regression detection | Establish baseline, run current version, compare metrics |
| Custom metrics | Understand scorer interface, choose pattern, test |
| Judge alignment | Design judge, collect SME labels, align with MemAlign |
| Prompt optimization | Build dataset with expectations, run GEPA, promote |
